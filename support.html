Top FAQ
Bagaimana cara mendaftar untuk pertama kali?

Anda dapat mendaftar melalui menu Get Started

Apa yang dibutuhkan pertama kali untuk mendaftar Eversheet?

Anda perlu mendaftarkan informasi organisasi/perusahaan anda (jika anda akan mendaftar sebagai organisasi/perusahaan), atau mendaftarkan informasi personal anda (jika anda akan mendaftar sebagai personal). Form yang diperlukan dapat dilihat pada halaman Register.

Ada berapa role yang tersedia di Eversheet?

Terdapat 3 role yang tersedia di Eversheet. System Owner, Administrator, dan Reguler User.

Bagaimana jika saya ingin tahu cara menggunakan suatu fitur?

Anda dapat mengklik menu User Guide pada aplikasi (menu bar sebelah kanan atas), atau masuk ke menu Product Guide

Apa yang harus saya siapkan di awal setelah saya sudah masuk aplikasi dan ingin mulai menggunakan Eversheet?

Terdapat beberapa data yang harus disiapkan terlebih dahulu. Ketika user pertama kali masuk aplikasi Eversheet, akan muncul di Dashboardnya informasi data apa yang harus dibuat, diantaranya User Group, User, Account Group, Account, dan Contract Group.

Saya tidak dapat upload file dokumen kontrak saya, apa yang harus saya lakukan?

Hal tersebut dikarenakan Registry di komputer Anda kemungkinan terhapus oleh aplikasi lain, yang menyebabkan content type pada file yang ingin Anda upload jadi tidak dapat terbaca oleh sistem. Jadi, Anda harus menambahkan kode pengenal file pada komputer Anda supaya Anda dapat mengupload type file yang gagal tersebut. Anda dapat akses https://eversheet.com/files/article/content-type.zip untuk dapat menambahkan kode pengenal file pada komputer Anda.
